## USE
```cli
./main file-base-with-all-text-for-make-wordlist.txt file-for-output-wordlist.txt
```

## SOURCE
```bash
git clone https://github.com/Jul10l1r4/gomutation.git
cd gomutation
go build main.go # Or run with: go run main.go
```
